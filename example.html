Efecto de Escritura en el Título

Este proyecto demuestra un efecto simple pero visualmente atractivo que escribe dinámicamente un título en la pestaña del navegador. El script modifica el contenido de la etiqueta `<title>` en tiempo real, simulando un efecto de máquina de escribir con intervalos aleatorios entre cada carácter. Este efecto puede ser útil para añadir elementos interactivos y llamativos a aplicaciones web o para experimentar con actualizaciones dinámicas del título de la página.

Para usar este script, simplemente inclúyelo en tu archivo HTML y llama a la función typeTitle con el texto del título deseado. El script se encargará del resto, creando una animación de escritura suave y atractiva para el título de tu página web.
